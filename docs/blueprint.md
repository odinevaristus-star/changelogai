# **App Name**: changelogai

## Core Features:

- Repository Synchronization: Connect seamlessly with Git providers to import commit history and project metadata.
- AI Synthesis Tool: Use an AI tool to reason about commit message clusters and transform technical changes into meaningful feature summaries.
- Semantic Classification Tool: Leverage an AI tool to automatically categorize entries into categories such as Features, Improvements, or Bug Fixes.
- High-Fidelity Timeline: Present software updates in a clean, chronological feed designed for maximum readability.
- Markdown Portability: Export release notes in structured Markdown for documentation or release emails.

## Style Guidelines:

- Primary color: High-Precision Indigo (#5C77FF) to represent technological sophistication.
- Background color: Midnight Obsidian (#111218) for a sleek, dark-mode developer experience.
- Accent color: Lucid Blue (#A3D8FF) to draw attention to call-to-actions and interactive status indicators.
- Headlines are set in 'Space Grotesk' (sans-serif) for a tech-focused, geometric aesthetic.
- Body text uses 'Inter' (sans-serif) for a clean, neutral look that emphasizes technical content.
- Monolinear glyphs with crisp angles, reflecting architectural code structures.
- Minimalist, focused center-aligned column layout that prioritizes content legibility.
